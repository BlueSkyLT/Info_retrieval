# Usage

place the *DATASET_DIRECTORY* under ```Infor_retrieval/```

```cd Info_retrieval```

```python indexing.py DATASET_DIRECTORY```

to search the first *NUMBER_OF_RESULTS* documents containing *QUERY_TERM*:

```python search.py QUERY_TERM NUMBER_OF_RESULTS```
